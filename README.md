# Task Manager API (Go)

A simple REST API built with Go for managing tasks.  
This project demonstrates CRUD operations, clean architecture, and JSON file persistence.

---

## 🚀 Features

- Create tasks
- View all tasks
- View single task
- Update tasks
- Delete tasks
- JSON file persistence

---

## 🛠️ Tech Stack

- Go (Golang)
- net/http package
- JSON handling
- File storage
