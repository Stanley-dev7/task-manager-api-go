Task Manager API (Go)

A simple REST API built with Go for managing tasks.  
This project demonstrates CRUD operations, clean architecture, and JSON file persistence.

 FEATURES
1. Create tasks
2. View all tasks
3. View single task
4. Update tasks
5. Delete tasks
6. JSON file persistence
TECH STACK
1. Go (Golang)
2. net/http package
3. JSON handling
4. File storage
5. Project Structure
